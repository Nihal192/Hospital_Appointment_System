# Hospital Appointment System

This repository contains a simple hospital appointment booking system with a Spring Boot backend and a Next.js frontend.

## Backend
- Located in the `backend` folder.
- Run with `./gradlew bootRun` (dependencies must be installed).
- Provides `/appointments` API for creating and listing appointments.

## Frontend
- Located in the `frontend` folder.
- Run with `npm install` then `npm run dev`.
- Connects to the backend at `http://localhost:8080`.
